# Web Vulnerability Scanner

A basic Python-based web vulnerability scanner that detects common security issues such as Cross-Site Scripting (XSS) and SQL Injection.

## 🚀 Features
- Detects reflected XSS vulnerabilities
- Identifies potential SQL Injection points
- Simple and lightweight scanning
- Easy to use for beginners

## 🛠️ Technologies Used
- Python
- requests library

## 📦 Installation

1. Clone the repository:
   git clone https://github.com/yourusername/web-vuln-scanner.git

2. Navigate into the folder:
   cd web-vuln-scanner

3. Install dependencies:
   pip install -r requirements.txt

## ▶️ Usage

Run the scanner:
python web_scanner.py

Enter a target URL when prompted:
Example: http://testphp.vulnweb.com
